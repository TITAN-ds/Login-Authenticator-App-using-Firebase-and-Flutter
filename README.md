# Flutter Firebase Auth App

A minimal yet powerful authentication app built with Flutter (Dart), integrated with Firebase Authentication and Database.

This app provides a smooth onboarding experience with Sign Up, Sign In, and Welcome screens, dynamically handling user sessions and securely fetching user data from Firebase.

##Features

Firebase Authentication (Email/Password)

Sign In & Sign Up flow

Real-time user data fetching from Firebase

Clone the repository

Add your Firebase project configuration

Enable Authentication in Firebase Console

Run the app on emulator or device
